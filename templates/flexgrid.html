Explanation of CSS elements

  • Margin: space outside the element
  • Padding: space inside the element
  • Width/height: the sizing and how big an element is
  • Border: line around an element
  • Colour: text colour
  • Background colour: background colour of an element

When the window becomes smaller, the flexbox gallery automatically adjusts and the boxes move to the next line. The layout is responsive because Flexbox distributes extra space automatically.

Eplanation of CSS elements

  • Static: default position
  • Relative: element stays in the normal flow but can be moved slightly
  • Absolute: element is removed from the normal flow
  • Fixed: element is positioned relative to the browser
  • Sticky: element behaves like relative until you scroll
  • Inherit: element takes the same position value as its parent element

The grid layout keeps items aligned in a fixed structure. When the window becomes smaller, the items will adapt but remain in the same rows and columns.
